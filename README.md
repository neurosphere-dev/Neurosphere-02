# Neurosphere-02
Neurosphere-02: The Autonomous Backbone (Tulang Punggung Otonom)
---
### Core Value Identity (TM)
Neurosphere-02 is the server-side infrastructure for **Technology Money (TM)**. It manages three core dialects:
* **ENPE (Engine)**: Native engine value. Total supply: 100 Trillion [cite: 2025-12-20].
* **LUV (Lovely Coin)**: Social and gift distribution [cite: 2025-12-23].
* **IND-EUR (Indienation Coin)**: Stable value anchor [cite: 2025-12-20].

### Autonomous Protocols
This system automatically enforces the following rules:
* **Locking**: 2-year lock for public assets and 3-year lock for founder's allocation [cite: 2025-12-20].
* **LUV Distribution**: 1% allocation for 1 million people (1 million LUV each) taken from founder's pool [cite: 2025-12-20].
* **Donation & Reserve**: 15% Donation Pool and 10% Reserve are open for immediate emergency use [cite: 2025-12-20].

*Status: Independent and Parallel Infrastructure (Non-Termux).*
